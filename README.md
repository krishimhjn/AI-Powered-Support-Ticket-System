# AI Support Ticket System (Version 1)

A backend support ticket management system built with **FastAPI**, **PostgreSQL**, and **SQLAlchemy**. It provides secure authentication, role-based access control, and ticket management through REST APIs.

## Features

* JWT Authentication
* User Registration & Login
* Customer & Agent Roles
* Create and Manage Support Tickets
* Role-Based Authorization
* PostgreSQL with SQLAlchemy ORM

## Tech Stack

* Python
* FastAPI
* PostgreSQL
* SQLAlchemy
* Pydantic
* JWT
* Passlib (bcrypt)

## APIs

### Users

* Register
* Login
* Get Current User

### Tickets

* Create Ticket
* Get My Tickets
* Get All Tickets (Agent)
* Get Ticket by ID
* Update Ticket Status (Agent)
